Software Projects

I've been a competent Python programmer for many years, but mostly worked in data-scientific roles, where there tends to be less emphasis on good software – partly because each employee tends to work independently of the others. I find this frustrating. I've always tried to write clean code – if nothing else, it's incredibly helpful for referring back to previous work – and more recently I've begun writing test suites, using VCS much more actively to plan and develop features, and generally trying to write software in such a way as to enable collaboration.

I've also started teaching myself more languages and frameworks, and about more advanced Python. This website represents my learning journey in HTML, CSS and JS (still in its infancy, but it's a work in progress). Additionally, you can see some examples of personal projects I've been working on in my GitHub account. I've written more about some of these in the posts that you can find linked in the sidebar or below.
